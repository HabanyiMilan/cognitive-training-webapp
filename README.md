# Cognitra (A Cognitive Training Web App)
Cognitive performance web application designed to improve and analyze users' cognitive abilities through interactive tasks and statistical evaluation.

## Project Overview
- Provides cognitive training exercises
- Tracks user performance
- Generates statistical insights

## Technology
### Backend
- Flask
- Flask-SQLAlchemy
- SQLite
- Authlib (OAuth2 - Google authentication)
- OpenAI API (AI-generated feedback)

### Frontend
- React
- Axios
- React Router

## Core Features
- User authentication
- Lifestyle Assessment Module (sleep, stress, physical activity, screen time)
- Task selection dashboard
- Time-based cognitive exercises with real-time score calculation
- Performance analytics dashboard
- AI-generated textual feedback
